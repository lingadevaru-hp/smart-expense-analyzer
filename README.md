## Project Description
Smart Expense Analyzer is an AI-powered expense tracking and 
analysis tool that helps users monitor, categorize, and derive 
insights from their daily spending patterns. The system leverages 
Python-based machine learning models to automatically classify 
expenses and surface actionable financial insights through a 
clean, responsive web interface.

---

## Key Features
- AI-powered automatic expense categorization
- Visual spending analytics and trend reports
- Income vs expense comparison dashboard
- Multi-category expense breakdown
- Downloadable expense reports
- Responsive and intuitive web interface

---

## Tech Stack
- **Frontend:** React.js
- **Backend:** Python, FastAPI
- **ML Models:** Scikit-learn, Pandas, NumPy
- **Database:** MongoDB
- **Authentication:** JWT

## Setup / Execution Steps

### Prerequisites
- Python 3.8+
- Node.js 16+
- pip
- MongoDB

### Installation

```bash
# Clone the repository
git clone https://github.com/lingadevaru-hp/smart-expense-analyzer
cd smart-expense-analyzer

# Install backend dependencies
pip install -r requirements.txt

# Install frontend dependencies
npm install

# Start backend
python app.py

# Start frontend (in a new terminal)
npm start
```

### Access
Navigate to `http://localhost:3000` in your browser.
